MCPBot Data Exports

Here you will find daily and stable exports of the MCP CSV files from MCPBot.

Base Folder: Semi-live CSV files exported from the bot database every 10 minutes. These contain the latest changes.
Snapshot Folders: Daily exports of the latest mapping changes.
Stable Folders: Exports of approved mappings when they are committed.
*nodoc Folders: Same CSV files as their non-nodoc named counterparts, except for that they do not contain the comments.
